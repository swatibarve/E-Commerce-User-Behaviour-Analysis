# E-Commerce-User-Behaviour-Analysis
A SQL-based funnel analytics project that tracks 5,000 users across a complete e-commerce journey — from first page view to final purchase — using BigQuery and custom DAX-style conversion metrics.

This project analyzes real-world style e-commerce event data to answer one core business question:
"Where do we lose customers, and which traffic sources actually convert?"

The dataset covers 9,381 events across 5,000 unique users over a 35-day period (Dec 30, 2025 – Feb 3, 2026), tracking every step of the purchase journey across four traffic sources.


 Tools & Skills

SQL: Google BigQuery SQL
Key Functions Used: COUNT(DISTINCT CASE WHEN ...), TIMESTAMP_DIFF, DATE_SUB, HAVING, CTEs (WITH)
Analytical Techniques: Sales funnel analysis, cohort-style time analysis, channel attribution
